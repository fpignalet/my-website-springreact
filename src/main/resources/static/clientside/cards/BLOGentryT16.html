Multi canvas instances

Solution to have many instances of the game canvas from the previous sandbox

  • create a Multiple class like the one on the left
  • Apply the following initialisation sequence for any other class
    class Input extends Multiple { ... static IDENT() { return "input" }; ... constructor() { super(Input.IDENT());
  • Reference related html items with an index number
    ... canvas id="canvas0" ...
  • Access all objects referenced like this using:
    ... const input = Multiple.get(Input.IDENT(), this.index); ...
