Some important points to develop with raw Javascript

  • It's possible to be pure OOP, using only classes exactly like other langages
  • Classes need to be exported like this:
    ... export default Multiple; ...
  • Classes need to be imported like this:
    < script type="module" > import { CLISIDE_LOADER } from "./clientside/clientside.js"; < /script >
    It's important to specify the script type="module", otherwise import / export return error.
    It's important to specify the ".js" postfix, otherwise there is sometimes loading issues.
